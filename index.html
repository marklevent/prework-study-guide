Prework Study Guide

HTML

  • The head element contains information about the webpage include metadata which is not typically visable to the user.
  • The body element represents the visible content shown to the user.
  • The footer element shows content such as copyright information.

CSS

  • CSS is the styling stage of developing a website.
  • CSS stands for casscading style sheets.
  • It determines how a website looks visually.

Git

  • Git is open source software for distributed version control.
  • Git allows users to branch off the main website code to work on individual elements of the code without affecting the main code.
  • Once users have added a feature or made changes to code, they can push this to Github for review by the team.
  • Once reviewed by the team, they can add comments, suggest changes or merge the branch into the main code which is the desired end outcome.

JavaScript

  • JavaScript is code which adds complex functionality to a website, such as interactive maps, videos, etc.
